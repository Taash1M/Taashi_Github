# Platform Modernization Case Study

## Context
Legacy OBIEE, slow reports, manual processes, and limited self-service.

## Approach
- Designed Azure + Databricks + ADLS architecture.
- Implemented ingestion + bronze/silver/gold layering.
- Standardized modeling with dbt and orchestrated with ADF/Airflow.

## Impact
- 70% reduction in report latency.
- 3x increase in adoption across 200+ stakeholders.
- Established a reusable foundation for future analytics use cases.
